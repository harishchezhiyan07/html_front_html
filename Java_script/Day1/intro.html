Variables

1. What is a Variable?

A variable is a container used to store data values in a program.

2. What is Declaration?

Declaration means creating a variable by specifying its name.

3. What is Initialization?

Initialization means assigning a value to a variable.

4. Three Types of Keywords Used for Variables in JavaScript

var – Old way of declaring variables.

let – Modern way, value can be changed.

const – Constant value, cannot be reassigned.

5. What is a Compiler in JavaScript and Its Name?

A compiler is software that translates source code into machine-readable code.

JavaScript engines use a Just-In-Time (JIT) Compiler.

Popular JavaScript engines:

V8 – Used in Google Chrome and Node.js

SpiderMonkey – Used in Mozilla Firefox

JavaScriptCore – Used in Safari
