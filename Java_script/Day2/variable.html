Datatypes List out that in java script

Data Type Description Example
String Stores text "Hello"
Number Stores integers and decimals 10, 3.14
Boolean Stores true or false values true, false
Undefined Variable declared but not assigned a value let x;
Null Represents an empty value "let y = null;"
BigInt Stores very large integer 67890n
Symbol Creates unique identifiers Symbol("id")

EXAMPLE

let name = "Harish"; // String

let age = 21; // Number

let isStudent = true; // Boolean

let city; // Undefined

let address = null; // Null

let bigNum = 123456789n; // BigInt

let id = Symbol("id"); // Symbol
