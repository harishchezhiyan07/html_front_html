Unit in css

1.em(element based):Is used for Component-specific sizing (e.g., padding inside a button that scales with its text size). 2.px is mainly used for Borders, shadows, exact spacing, or fixed-size images. 3.rem(root element):font size of root element or mainly typography and layout. 4.% :the size of parent element property .fluid layout dimension like width or height.

RGB, RGBA, and HEX color values in CSS

1.RGB-uses decimal number(0-255)for red,green and blue 2.RGBA-adds an alpha channel for transparency (0.0 to 1.0) 3.HEX-use hexadecimal shorthand(0-9,A-F) to represent RGB values.
